Visual indicators of ticket availability for various performances for theaters using Audience1st to sell tickets.

== Description ==
The plugin displays color \"thermometers\" to indicate excellent availability (green), limited availability (yellow), and almost sold out (red) for various performances, for theaters using the open-source Audience1st boxoffice and backoffice system for ticket sales.  Settings can be customized in the "Appearance" menu of the WordPress admin screen.  Originally authored by Denise Beyer, Shane Rogers, and Anne Stefanyk of Kanopi Studios; packaged as a WordPress plugin and maintained by Armando Fox.
